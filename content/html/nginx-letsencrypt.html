Letsencrypt with nginx

Preface

Letsencrypt is a service/tool which allows you to generate free,trusted, ssl certificates. On it's own, it only generates the certificates and does nothing more. Assuming we've installed the necessary packages, we can use "certbot" and it's nginx plugin to automatically configure our web server with SSL.

Installation

apt install certbot python-certbot-nginx
